🔍 Understanding for...in vs for...of

👨‍💼 Example 1: for...in on Object (recommended)

const person = { name: "John", age: 25, city: "New York" };

for (let key in person) {
  console.log(key + " 👉 " + person[key]);
}

🍎 Example 2: for...in on Array (not recommended)

const fruits = ["🍎", "🍌", "🍇"];

for (let index in fruits) {
  console.log(index, fruits[index]);
}

🍇 Example 3: for...of on Array (recommended)

const fruits = ["🍎", "🍌", "🍇"];

for (let fruit of fruits) {
  console.log(fruit);
}

💬 Example 4: for...of on String

const word = "HELLO";

for (let char of word) {
  console.log(char);
}
